# Prompts I Used

These are the kinds of questions I asked while building and improving the project.

## 1. Understanding the task
Can you break down the Prodesk IT Week 1 assignment and explain what exactly is required for Level 3?

## 2. Project structure doubt
I do not want to submit just one HTML file. What would be a clean project structure for a Vite + Tailwind landing page?

## 3. Design direction help
Suggest a design style for a digital agency landing page that feels modern, professional, and natural instead of looking like a template.

## 4. Navbar idea
How can I make a sticky responsive navbar with a logo, navigation links, a mobile menu, and a dark mode toggle?

## 5. Glass effect help
How can I create a frosted glass navbar using `backdrop-filter` without making the text hard to read?

## 6. Hero section writing help
Give me a few headline and subheadline ideas for a digital agency website called Prodesk IT. I want the copy to sound simple and real.

## 7. Service section help
Help me write short content for three services: SEO, Web Development, and Digital Marketing.

## 8. Dark mode doubt
Show me a simple JavaScript approach for dark mode toggle with theme saved in localStorage.

## 9. Interaction polish
What are some small hover effects I can add to buttons, cards, and nav links so the page feels polished but not too flashy?

## 10. Accessibility check
Can you review the structure and tell me what I should improve for semantic HTML, keyboard navigation, labels, and focus states?

## 11. Lighthouse preparation
What should I check before running Lighthouse so I have a better chance of getting a very high performance and accessibility score?

## 12. README help
Help me write a simple README for this internship project with setup steps, features, and deployment instructions.

## 13. Final review
Can you review the full project and tell me if anything still feels too robotic, too generic, or unfinished?
